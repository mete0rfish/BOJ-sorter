# BOJ 리스트 정렬 도우미

![Version](https://img.shields.io/badge/version-1.0-blue)

백준 온라인 저지(BOJ) 리스트 페이지의 테이블을 클릭으로 정렬하는 크롬 확장 프로그램입니다.

## ✨ 기능

*   백준 온라인 저지(BOJ)의 문제 리스트, 게시판 리스트 등 테이블 형태의 데이터를 클릭하여 원하는 기준으로 정렬할 수 있습니다.

## 🚀 설치 방법

1.  이 저장소를 클론(Clone)하거나 ZIP 파일로 다운로드합니다.
    ```bash
    git clone https://github.com/your-username/BOJ-sorter.git
    ```
    (또는 GitHub에서 'Code' 버튼을 클릭하여 'Download ZIP'을 선택합니다.)

2.  크롬 브라우저를 엽니다.

3.  주소창에 `chrome://extensions`를 입력하여 확장 프로그램 관리 페이지로 이동합니다.

4.  우측 상단의 '개발자 모드'를 활성화합니다.

5.  '압축 해제된 확장 프로그램을 로드합니다.' 버튼을 클릭합니다.

6.  클론하거나 다운로드한 프로젝트 폴더(예: `BOJ-sorter`)를 선택합니다.

## 💡 사용 방법

1.  백준 온라인 저지(BOJ)의 문제 리스트 페이지나 게시판 페이지에 접속합니다.
2.  테이블 헤더(예: 문제 번호, 제목, 정답률 등)를 클릭하여 원하는 기준으로 데이터를 정렬합니다.

프로젝트 개선에 기여하고 싶으시다면 언제든지 환영합니다!

1.  이 저장소를 포크(Fork)합니다.
2.  새로운 기능 또는 버그 수정을 위한 브랜치를 생성합니다.
    ```bash

    ```
    (예: `feature/add-dark-mode` 또는 `fix/wrong-sorting-logic`)
3.  변경 사항을 커밋합니다.

    ```bash
    git push origin feature/your-feature-name
    ```
5.  원래 저장소로 풀 리퀘스트(Pull Request)를 생성합니다.
